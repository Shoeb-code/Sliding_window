---

## 📚 What is Sliding Window?

The **Sliding Window** technique is an optimization approach used to solve problems involving arrays and strings by maintaining a window of elements and updating it efficiently instead of recalculating values repeatedly.

### Types of Sliding Window

* ✅ Fixed Size Sliding Window
* ✅ Variable Size Sliding Window
* ✅ Dynamic Sliding Window
* ✅ Two Pointer Technique
